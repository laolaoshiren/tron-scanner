# Tron 交易扫描器

一个用于扫描 Tron 区块链上带备注的交易的工具。支持多种代币类型，可以按地址筛选，支持备注过滤，并且可以扫描大范围区块。

## 主要功能

1. **多网络支持**
   - 主网
   - Shasta 测试网
   - Nile 测试网

2. **多代币类型**
   - TRX（波场原生代币）
   - TRC20
   - TRC721（NFT）
   - TRC1155
   - TRC10

3. **高级筛选功能**
   - 备注关键词过滤（支持多关键词）
   - 钱包地址筛选（支持多地址）
   - 支持筛选发送方/接收方

4. **灵活的扫描选项**
   - 支持自定义区块范围
   - 支持从大到小/从小到大扫描
   - 快捷选择最近的区块范围（100-100万）
   - 实时显示扫描进度
   - 可随时停止扫描

5. **用户友好特性**
   - 自动保存所有设置
   - 交易哈希和地址链接到区块浏览器
   - 美观的进度显示
   - 完整的时间戳显示

## 技术栈

- Vue 3
- Element Plus
- TronWeb
- Vite

## 安装和运行

1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
npm install
```

3. 运行开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

## 使用说明

1. **选择网络**
   - 根据需要选择主网或测试网

2. **设置扫描范围**
   - 输入起始和结束区块
   - 或使用快捷按钮选择最近的区块范围

3. **选择代币类型**
   - 可多选不同类型的代币交易

4. **设置过滤条件**
   - 输入需要过滤的备注关键词
   - 输入需要筛选的钱包地址

5. **开始扫描**
   - 点击"开始扫描"按钮
   - 可以随时点击"停止扫描"中断操作

## 注意事项

1. 为保证性能，单次最多扫描 100 万个区块
2. 所有设置会自动保存在浏览器中
3. 建议在扫描大范围区块时使用更多的过滤条件
4. 如果遇到网络问题，可以尝试切换节点或重试
